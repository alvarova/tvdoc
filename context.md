# Contexto

## Versiones
- **Node.js:** 20 LTS
- **React:** 18
- **Vite:** 4
- Código escrito en **JavaScript** con módulos ES.

## Buenas prácticas
- Utilizar componentes funcionales y hooks de React.
- Mantener la estructura modular en `components`, `hooks`, `services` y `utils`.
- Respetar principios **SOLID** y optimizar el rendimiento con carga eficiente.
- Documentar cada modificación y mantener el `README` actualizado.
- Ejecutar `npm run lint` y `npm run build` antes de subir cambios.
